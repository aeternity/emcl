# emcl

[![CircleCI][circleci badge]][circleci]
[![License][license badge]][license]
[![Build Tool][build tool]][rebar3]

An Erlang wrapper for the BLS12-381 curve API of [mcl](https://github.com/herumi/mcl)

## Build

    $ rebar3 compile

## Unit tests

    $ rebar3 eunit
